Les langages de programmation !

Comment a été faite cette page ?

Cette page a été codée en HTML, le langage du web. Cet acronyme signifie Hypertext Markup Language, ou Langage de Balisage Hypertexte.

Cependant, une simple page faite en HTML n'est pas très jolie. Pour la styliser un peu, il faut utiliser un autre langage : le CSS. Le CSS, c'est l'abréviation de : Cascading Stylesheet, et en français : Feuilles de Style en Cascade.

Grâce à ce langage, les pages web sont beaucoup plus jolies. C'est beaucoup plus pratique que de tout faire manuellement en HTML.

Langages de programmation du web

Pour rendre des sites web dynamiques, il y a d'autres langages à connaître :

  • Le plus connu est sûrement le JavaScript, parfois appelé JS, JScript ou ActionScript.

    Il s'agit d'un langage de programmation (contrairement au HTML et au CSS), utilisé pour rendre des pages web interactives, bien que le CSS sache plus ou moins le faire. Il est aussi employé pour les serveurs, notamment avec Node.js ou encore Deno.

    Aussi, il est possible de créer un bot Discord. Je ne vais pas expliquer comment faire ceci, mais cliquez ici pour plus d'infos.

  • Il y a un langage pour créer des sites web dynamiques : le PHP.

    Ce langage est utile pour communiquer avec des bases de données, et faire des systèmes de connexion sur son site. Il permet aussi de réutiliser des éléments d'une page pour y mettre sur tout le site.

    Bien que ce langage soit très pratique il ne peut pas s'exécuter comme ça sur son navigateur, contrairement au HTML, au CSS et au JavaScript.
    Il faut installer un serveur virtuel sur son PC pour pouvoir commencer à coder son site avec. Aussi, ce langage n'est pas très utile sans base de données (MySQL par exemple).

  • Enfin, il est possible d'utiliser un framework pour faire des sites dynamiques : ASP.NET permet de générer des pages dynamiques !
    Le ASP.NET est assez dur à comprendre (de plus, je n'en ai jamais fait), c'est pourquoi je n'en parlerai pas énormément.
